Yokai Horror

[ View Source on GitHub ]

Yokai Horror was recently started by myself and a team of friends after graduating from college. As of now, there are only some short GIFs of stealth mechanics, shaders, and some character concepts done. Even in the short time working on it so far, I've learned a lot! This project is what got me started with the Godot engine, and it's been very enjoyable to encorporate low-level programming in the project using C++ via Godot 4's GDExtension features. I hope to expand on this project very soon, once our team is ready!
