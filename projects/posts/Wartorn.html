Wartorn

[ Download on itch.io ]

I helped to develop "Wartorn" as a technical artist as part of a class project at UCI. I utilized scripts and software to procedurally generate models for buildings and other set pieces, as well as procedurally create destructed versions of those models. In addition, I created various models, rigs, scenes, and shaders by hand for the game.
