gawk-trees

[ View Source on GitHub ]

"gawk-trees" is a project I started working on initially as a sort of "challenge" project for one of my classes as a senior in college, centered around learning UNIX-based operating systems, but has since developed into something I've had the opportunity to be hired part-time to work by my professor in an effort to help improve the logistics of his research. Professor Hayes' research, as well as the work as many others, requires the parsing of huge amount of data, and the tool of choice for those like my professor is AWK. This extension seeks to provide a full interface for working with binary trees and hierarchical binary trees as the quintessential data structure for an AWK program as opposed to the usual hash maps under the hood for its associative arrays. The project is still currently in progress, but the memory savings have proven to be up to 4x when compared to using AWK's standard arrays.
