Discord Accountability Partners

[ View Source on GitHub ]

I created this Discord bot in Rust as a fun way for me and my partner to help keep each other accountable for the tasks we want to complete for the day, week, or month! This project was also a fun way for me to learn Rust, which I decided since an online application handling personal information should probably use a language designed for security, but along the way I also learned more about using popular APIs such as Discord's and even some cryptography. Originally, the method to store persistent info about users and tasks was going to be handled through encrypted JSON files, but I shifted over to using Discord messages as persistent data, although information like the bot token and channel IDs still have to be encrypted of course. You can read more and check out the source code at the link above!
